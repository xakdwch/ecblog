# 概述

Etcd-Carry是一个用于etcd集群之间数据迁移的工具，可以将需要做迁移的etcd集群数据复制到世界任意目标etcd集群。

Etcd-Carry不仅可以对整个集群数据进行迁移，也可以自定义规则对指定数据进行迁移。Etcd-Carry部署简单灵活，支持二进制部署和容器部署。
